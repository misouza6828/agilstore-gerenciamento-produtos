# AgilStore - Gerenciamento de Produtos

Aplicação desenvolvida em Node.js para gerenciamento de produtos de uma loja de eletrônicos, com foco em automação de inventário e persistência de dados.

## Funcionalidades
- Adicionar produtos
- Listar produtos
- Atualizar produtos
- Excluir produtos
- Buscar produtos por ID ou nome
- Persistência automática em arquivo JSON

## Tecnologias Utilizadas
- Node.js
- JavaScript
- Módulos nativos (`fs`, `readline`)
